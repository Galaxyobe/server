# FRP

[Github](https://github.com/fatedier/frp)


## 下载

> wget https://github.com/fatedier/frp/releases/download/v0.13.0/frp_0.13.0_linux_amd64.tar.gz
> tar xvf frp_0.13.0_linux_amd64.tar.gz

## 配置服务器

* 安装

> ./install server [-d]

* 卸载

> ./uninstall server [-d]

## 配置客户端


* 安装

> ./install client [-d]

* 卸载

> ./uninstall client [-d]
